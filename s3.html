Module 3: Image Classification & Analysis

Understanding classification and interpretation of satellite images

Course Content

1. Supervised Classification

Supervised classification is a technique where the user provides training data to classify satellite images. The analyst selects sample pixels for each class such as water, vegetation, or urban areas, and the system uses these samples to categorize the rest of the image.

2. Unsupervised Classification

In unsupervised classification, the system automatically groups pixels into clusters based on their spectral properties. The user then interprets these clusters and assigns them to meaningful classes.

3. Accuracy Assessment

Accuracy assessment is used to evaluate how correctly the classification has been performed. It compares classified data with reference data to measure reliability and performance.

4. Applications

Image classification is widely used in land use mapping, agriculture monitoring, forest management, urban planning, and disaster analysis.
