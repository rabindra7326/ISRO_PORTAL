Satellite Image Processing

Understand image enhancement & analysis.

Course Overview

This course introduces the basic concepts of remote sensing, including satellites, sensors, and their applications. It is designed for beginners.

Course Modules

Module 1: Introduction to Image Processing

Definition of Image Processing, Types of Images, Digital Image Basics, Importance

Module 2: Image Enhancement Techniques

Contrast Enhancement, Filtering Techniques, Noise Removal, Image Correction

Module 3: Image Classification & Analysis

Supervised Classification, Unsupervised Classification, Accuracy Assessment, Applications
