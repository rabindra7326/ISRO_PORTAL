Module 2: Spatial Data & Models

Understanding GIS data structures and sources

Course Content

1. Spatial Data

Spatial data represents the geographic location and characteristics of features on Earth. It includes both location (coordinates) and attribute information.

2. Vector Data

Vector data represents real-world objects using points (locations), lines (roads), and polygons (areas like lakes or boundaries).

3. Raster Data

Raster data is made up of pixels arranged in a grid format. It is commonly used for satellite imagery and elevation data.

4. Data Sources

Spatial data can be collected from satellites, GPS, field surveys, drones, and existing digital databases.
