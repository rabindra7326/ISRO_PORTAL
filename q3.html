Module 3: GIS Analysis in QGIS

Learn analysis tools and map creation

Course Content

1. Buffering

Buffering creates zones around features such as roads or rivers. It helps analyze nearby areas and impacts.

2. Overlay Analysis

Overlay combines multiple layers to produce new insights. It is useful for decision-making and spatial planning.

3. Map Creation

QGIS provides layout tools to design maps with legends, scale, and titles.

4. Exporting Maps

Maps can be exported as PDF or images for sharing and printing.
