Module 2: Data Handling in QGIS

Learn vector, raster and layer management

Course Content

1. Vector Data

Vector data represents geographic features using points, lines, and polygons. It is used to represent discrete features such as roads, rivers, and boundaries.

2. Raster Data

Raster data is made up of pixels or grids and is commonly used for satellite images and elevation models. Each pixel contains a value representing information.

3. Layer Management

QGIS allows multiple layers to be added, removed, and organized. Users can control visibility, styling, and order of layers.

4. Attribute Table

The attribute table stores information about spatial features. It helps in analyzing and editing data easily.
